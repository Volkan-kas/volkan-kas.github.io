Begin

Bij HTML begin je altijd met hetzelfde stukje tekst. Kopieer dit om te beginnen:

<!DOCTYPE html>
<html lang="nl">
<head>
 <meta charset="utf-8">
 <title></title>
</head>
<body>
</body>
</html>

Als je dit hebt gedaan, kun je verder door in de zijbalk te klikken op tag die je nodig hebt.

H1, H2, p en ul

Deze tags worden gebruikt voor het opmaken van tekst. <h1> en <h2> om de tekst groter te maken, <p> voor paragraven en <ul> voor een punt voor de zin. Probeer ze uit om te kijken wat ze doen.

Hier enkele voorbeelden met hoe ze te gebruiken en wat ze doen:

<h1>Deze tekst wordt groot</h1>
<h2>Deze tekst wordt iets minder groot</h2>
<p>Dit wordt na meerdere keren gebruiken, paragraven</p>
<ul>Dit wordt een tekst met een punt ervoor</ul>

br

Dit is een tag die je vaak gebruikt. Dit voegt namelijk een enter toe in de tekst. <br> hoef je NOOIT te sluiten. Doe dit dus ook niet. Hier een voorbeeld met hoe het werkt:

Deze<br>tekst<br>staat onder<br>elkaar door<br>de brs.

img

Als je een foto of gif wil toevoegen aan je website, gebruik je <img>. door <width> of <height> kun je de grote aanpassen. Hier een voorbeeld:

<img src="(map met foto)/(naam).(extensie)" width="450px"/>

<img src="Images/voorbeeldfoto.jpg" width="450px"/>



table

<table> gebruik je om een tabel te maken. <tr> maakt nieuwe regels, en <td> nieuwe vakjes. Vergeet deze niet te sluiten! Hier een voorbeeld:

<table border="1">
 <tr>
  <td>linksboven</td><td>rechtsboven</td>
 </tr>
 <tr>
  <td>linksonder</td><td>rechtsonder</td>
 </tr>
</table>

style

<style> maakt websites mooi. Ze veranderen kleuren, fonts, grote, enz. Er zijn verschillende manieren om dit toe te voegen. Bijvoorbeeld in de <head>. Hier een voorbeeld.

<head>
 <meta charset="utf-8">
 <title></title>
 <style>
  body {
  background-color: #222222;
  font-family: arial;
  font-size: 16px;
  text-align: center;
  }
  .naam {
  color: green;
  font-family: fantasy;
  font-size: 30px;
  }
 </style>
</head>
<body>
 <p class="naam">Volkan</p>
</body>

a

Wil je een link toevoegen naar een andere website of pagina? dat kan! dit doe je met <a>. Je kan dit op tekst, fotos(img), buttons enz. Als je target _blank toevoegd, opent de link in een nieuwe pagina. Hier een voorbeeld:

<a href="google.com" target="_blank">Deze tekst opent google</a>

<a href="test.html" target="_blank">Deze tekst opent een test pagina</a>
